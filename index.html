CHEAT SHEET EMMET

Emmet, l’indispensable outil du développeur

Ce petit plugin va tout simplement vous permettre d’accélérer votre rythme de développement en passant outre toutes les étapes rébarbatives parfois présentes dans le développement. Si je vous dis, je vais créer un tableau HTML contenant 8 lignes de 5 cellules chacune, contenant elles aussi 8 liens chacune. Et si maintenant, j’ajoute que tout ce contenu HTML, je vais le générer grâce à une seule ligne de code?

Syntaxe

Child: >

nav>ul>li

<nav>
<ul>
<li></li>
</ul>
</nav>

Multiplications: *

Exemple

ul>li.item$*5

<ul>
<li class="item1"></li>
<li class="item2"></li>
<li class="item3"></li>
<li class="item4"></li>
<li class="item5"></li>
</ul>

ID and CLASS attributes

Exemple

#header

<div id="header"></div>

.title

<div class="title"></div>

form#search.wide

<form id="search" class="wide"></form>

p.class1.class2.class3

<p class="class1 class2 class3"></p>

html

Exemple

a
<a href=""></a>
img
<img src="" alt="" />
input:s
<input type="submit" value="" />
link:css
<link rel="stylesheet" href="style.css" />
str
<strong></strong>
sect
<section></section>

Sibling: +

Exemple

div+p+bq

<div></div>
<p></p>
<blockquote></blockquote>

Climb-up: ^

Exemple

div+div>p>span+em^bq

<div></div>
<div>
<p><span></span><em></em></p>
<blockquote></blockquote>
</div>

div+div>p>span+em^^bq

<div></div>
<div>
<p><span></span><em></em></p>
</div>
<blockquote></blockquote>

Text: {}

Exemple

a{Click me}

<a href="">Click me</a>

p>{Click }+a{here}+{ to continue}

<p>Click <a href="">here</a> to continue</p>

CSS

Exemple

margin:auto;
bgc
